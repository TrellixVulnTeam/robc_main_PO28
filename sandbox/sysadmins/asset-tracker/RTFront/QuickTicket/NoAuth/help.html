How to create a QuickTicket

  • Connect to QuickTicket website (https://qt.gorillanation.com/)
  • Choose the category of the request, that will determine which group of users wil receive the request (Windows admins, Linux admins, Configuration manager)
  • Then choose between the other two dropdown menus to enter a more specific request.
    Example :
  • Once you have selected the specific category of your request fill in the requested informations. Be as precise as possible so we can answer your request in a timely manner
  • To validate and submit your request you must login using your Active Directory (windows) account
  • Upon submmission you will receive a confirmation email from the system
  • Reply to that email for any informations concerning your request
